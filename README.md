## 🎯 What is this?

A **lightweight sentiment analysis app** that detects emotions in **Arabic** and **English** text using AI. 

Simply type a sentence, and it tells you if it's positive, neutral, or negative with a 1-5 star rating.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🌐 Bilingual | Works with Arabic & English |
| ⭐ 5-Star Rating | 1⭐ = very negative, 5⭐ = very positive |
| 📱 Mobile Friendly | Runs perfectly on phones |
| 🚀 Cloud Based | No installation needed |
| 🔗 Shareable Link | Send the demo to anyone |
